Rel Control

Sistema em Flutter para controle de arquivos, desenvolvido para facilitar a organização e acesso rápido a documentos importantes.

Tecnologias utilizadas:

  • Flutter
  • Dart
  • Shared Preferences
  • Flutter Archive
  • PostgreSQL

Problema

Durante minhas atividades no suporte técnico responsável pelo módulo de relatórios de um sistema empresarial, identifiquei uma deficiência crítica na gestão dos arquivos de relatório — tanto nos templates padrão fornecidos pela empresa quanto nos modelos customizados desenvolvidos sob demanda.

O fluxo atual mantinha os relatórios exclusivamente no ambiente do cliente. Sempre que um novo relatório era criado ou um modelo existente era personalizado, o arquivo era editado e armazenado diretamente no servidor local do cliente. Esse processo apresentava um risco operacional significativo: em casos de falhas no servidor, corrupção de dados ou exclusões acidentais, os arquivos eram perdidos permanentemente, pois não existia um mecanismo centralizado de versionamento, backup ou sincronização em um repositório seguro.


Solução

O Rel Control permite que o técnico registre cada modificação realizada em um relatório, vinculando a edição à respectiva tela e opção de impressão onde o modelo é utilizado. Dessa forma, o sistema mantém um histórico centralizado que funciona como backup tanto para os relatórios personalizados de cada cliente quanto para os templates padrão da empresa, garantindo rastreabilidade, integridade e recuperação segura dos arquivos.
